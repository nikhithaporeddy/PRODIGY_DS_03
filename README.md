# 🚀 Customer Purchase Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a customer will purchase a product/service based on demographic and behavioral data using machine learning models.

The objective is to help businesses identify potential customers and improve marketing efficiency.

---

## 📊 Dataset
- Bank Marketing Dataset (UCI Repository)
- Features include:
  - Age
  - Job
  - Marital status
  - Education
  - Balance
  - Contact details
  - Campaign information
  - Previous interactions

---

## ⚙️ Workflow

### 1️⃣ Data Preprocessing
- Loaded dataset with correct delimiter (`;`)
- Encoded categorical variables using Label Encoding
- Saved cleaned dataset for model training

### 2️⃣ Model Building
- Decision Tree Classifier
- Random Forest Classifier

### 3️⃣ Model Evaluation
- Accuracy Score
- Classification Report (Precision, Recall, F1-score)
- Confusion Matrix

### 4️⃣ Visualization
- Confusion Matrix Plot
- Feature Importance Graph

---

## 📈 Results
- Achieved strong performance using Random Forest
- Model performs well on majority class
- Identified key features influencing customer decisions
